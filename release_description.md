## Alfred v0.1.2

This release continues Alfred's early local-first foundation work.

### Highlights
- Continued SQLAlchemy-based foundation work
- Improved project structure toward cleaner domain growth
- Person-context implementation progress
- Green test suite
- Updated vision to better reflect Alfred's development doctrine:
  - one concept at a time
  - one file at a time
  - full copy/paste file outputs preferred
  - smaller focused files in neat folder structures over monoliths

### Notes
Alfred remains an early-stage self-hostable household intelligence project focused first on durable local memory, clean architecture, and incremental growth.
